# Comment Créer Votre Premier Agent

Ce tutoriel vous guidera à travers le processus de création de votre premier agent dans Agentic Space.

## Prérequis

Avant de commencer, assurez-vous de :
- Avoir un compte sur Agentic Space
- Être authentifié dans le système

## Étape 1 : Accéder à la Zone de Création

1. Connectez-vous à votre compte
2. Dans le menu principal, cliquez sur "Créer un Agent"
3. Vous serez redirigé vers la page de création d'agents

## Étape 2 : Configurer les Informations de Base

Remplissez les champs suivants :

- **Nom** : Un nom descriptif pour votre agent
- **Description** : Une brève explication de ce que fait votre agent
- **Instructions** : Les directives que votre agent suivra

![Exemple de configuration](imagens/configuracao.png)

## Étape 3 : Définir les Capacités

Sélectionnez les capacités que votre agent aura :
- Traitement de texte
- Analyse de données
- Génération de contenu

## Étape 4 : Tester et Publier

Après avoir configuré votre agent :
1. Cliquez sur "Tester" pour vérifier le fonctionnement
2. Si tout est correct, cliquez sur "Publier"
3. Votre agent sera disponible à l'utilisation

## Conseils Supplémentaires

- Gardez les instructions claires et objectives
- Testez différentes capacités avant de publier
- Revoyez régulièrement les paramètres de votre agent

## Conclusion

Félicitations ! Vous avez créé votre premier agent dans Agentic Space. Continuez à explorer les fonctionnalités disponibles.
